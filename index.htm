What is PonyOS?

PonyOS is a new Unix-like operating system.

Is PonyOS a Linux distribution?

No! PonyOS uses its own kernel, built from scratch.

Using the Disk Image

The disk image can be used with QEMU. Other emulators may work but are not supported.

Extract the disk image from the tarball and then execute QEMU as follows:

qemu-system-i386 -m 1024 -hda ponyos-disk.img -vga std

(If you do not have sufficient RAM on your host, you may use -m 512 instead.)

The default user is local and the default password for that user is also local.

There is also a root user with password toor.

Building from Scratch

To build PonyOS from scratch, please use an Ubuntu or Debian-derived host.

Download and extract the code archive, or clone from our git repository and then run ./build.sh. Note that we do not support building PonyOS from scratch and this process may not work. Complete builds may take anywhere from thirty minutes to many hours, depending on your hardware.

Git Repository

PonyOS is developed through Github. You can find our repository at klange/ponyos.

FAQ

Can I put this on a bootable USB drive?

Unlikely, unless your motherboard supports IDE disk emulation for USB devices.

Does PonyOS work on real hardware?

Booting just fine on my Dell Mini 9.

Is this really not a Linux distro?

PonyOS uses a fully custom kernel, which is why it doesn't support things like networking or USB drives.

Does PonyOS come with a compiler?

No.
